Welcome onboard!

The Sarah Olofinro Library(SOL) is an physical and online library which treasures every source of knowledge from across generations and countries in print and non-print materials. Since 2000, we has been acquiring, consolidating, documenting, and restoring the library’s collections–building towards a web page for every book ever publised. To know more, view about us.

Rules of the library

To maintain the values the library provide for users, a few rules have been put in place by the organization. They are:

  • Only users registered with the library will be allowed to borrow library materials.
  • Some library collections are not for loan and they are not avaiable for loan on the website.
  • Late returns of borrowed books shall attract a penalty at a rate to be determined.
  • Any user who causes damage to library materials or other property will foot the cost of repair.
  • Only couriers in partnership with the library can be used for delivery.
